# Frontend - FAINANCE

This folder contains the mobile-first frontend for the FAINANCE POC.

## Run locally

```bash
npm i
npm run dev
```

## Build

```bash
npm run build
```

## App modules

- Dashboard
- Transactions
- Budget
- Goals
- Calendar
- Smart Input
- AI Advisor
- Reports

For full project context and POC alignment, see the repository root `README.md`.
